## 환경변수 관리
### 1. `.bashrc`에 환경변수 등록하기
1. `vi ~/.bashrc` 후 환경변수 추가 
```
# .bashrc 파일
...
...
export TELEGRAM_TOKEN='당신의토큰' # 토큰 정보 환경변수에 추가
```
2. `.bashrc` 다시 실행하기
```
source ~/.bashrc
```
3. Ruby 파일에서 불러오기
```
token = ENV['TELEGRAM_TOKEN']
```
